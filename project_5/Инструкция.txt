1) Установка XAMPP ( https://www.apachefriends.org/ )
запуск XAMPP Control Panel
Скопировать содержимое репозитория в папку C:\xampp\htdocs, 
если директория установки не изменялась.

В панели управления XAMPP нажмите "Config"
напротив запущенной MySQL БД.

Нажать "Start" на Apache и MySQL 
Нажать "Admin" на MySQL:
Откроется страница в браузере с phpMyAdmin,
Все обращения и пользователи хранятся в БД "violations".

2) Для удобного просмотра кода откройте папку 
с проектом в Visual Studio Code.

3) Чтобы запустить сайт в адресной строке в браузере введите:
http://localhost/NarusheniyamNet/login.php
открытие страницы авторизации.
Вход под юзером с логином 123 и паролем 123. Или создание нового.

Чтобы зайти под админом введите логин admin и пароль admin_pass.

4) Готово!
теперь вы можете составлять обращения под юзером 
и задавать им статус под админом.
